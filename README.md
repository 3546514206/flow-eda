# flow-eda

#### 介绍
一个由Java语言开发的基于事件驱动的流式低代码编程应用程序，您可以在编辑器中采用拖拽的形式来实现业务编程工作，一键部署，支持自定义组件和功能扩展。让开发和部署工作变得更简单高效。

项目名称介绍：
- flow: 流式编程
- eda: 事件驱动应用程序(Event Driven Application)
